class LinearModel:

    def __init__(self):
        self.pmega = 0
        self.b = 0
        
    def fit(self,x_train,y_train):
        self.omega = (y_train[0]-y_train[1]) / (x_train[0] - x_train[1])
        self.b = y_train[0] - x_train[0]*self.omega
        return
    
    def predict(self,x_test):
        y_test = self.omega * x_test + self.b
        return y_test

1. 2 随机模型机器学习算法¶

  1. 样本数据存在随机的观测误差，则并非上一节那么理想
  2. 采集一组数据，m个数据点。这组数据叫样本或样本集，m叫做样本容量
  3. f(x;w,b) = wx + b ,根据这组数据选择最佳的w和b，称为最佳参数
  4. 选择最优模型的方法：a.最小二乘法 b.极大似然估计
  5. 经典的数据分析方法：回归分析

1.2.1 最小二乘法¶

  1. 所谓的损失函数就是样本集所有样本点误差之和
  2. 不同的损失函数，实际上是误差定义的方式不同
  3. 最小二乘法在选择误差时选择的是(yi-f(x,theta))^2
  4. 采用平方形式的误差，损失函数为实数集上的凸函数，凸函数之和为凸函数。凸函数存在唯一的极值点，且是最小值点。因此可以用微积分中求极值的方法进行最小化求解。

1.2.2 极大似然估计¶

  1. 为随机模型，$Y=f(X)+ \varepsilon$
  2. 获得x出现时y出现的概率密度函数$p(y|x)$ ，含f(x)的参数$\theta$
  3. 根据样本集中的各点(xi,yi),写出每个点出现的概率$pi = p(yi|xi)$,并相乘，就是这个点集出现的概率
  4. 使得点击出现概率最大的$\theta$便是最优的参数

1. 3 随机变量与数学语言¶

1.3.1 随机变量¶

  1. 观测过程存在误差，故将随机性引入机器学习模型
  2. 合并次要因素以简化模型,不重要的参数可以合并为一个参数。理论依据：中心极限定理，许多小影响的独立随机因素带来的总影响可看作一个近似服从正态分布的随机变量。
  3. 研究总体的规律需要使用随机模型。

1.3.2 随机变量应用举例¶

  1. 随机函数：一组形式相同但是参数不同的函数的集合。
  2. 其应用：房屋市场价的随机性可以认为是因房屋中介公司使用的报价模型不一样

1.3.3 数学语言¶

  1. 凸函数琴生不等式：f(E(X))≤E（f(X)）
  2. 凸函数：定义。凸函数上任意两点的割线位于函数图形的正上方。
  3. 凸集：任意两点线段上的点仍属于D
  4. 凹函数琴生不等式：f(E(X))≥E（f(X)

1. 4 更加复杂的数学模型¶

1.4.1 数学形式未知的模型¶

  1. 模型假设对机器学习至关重要，模型假设错误则后续工作变得毫无意义。
  2. 科学的建模方法：数据可视化建模、数学建模

1.4.2 多元模型¶

  1. 由多元函数表示的模型就是多元模型
  2. 这么多个自变量就叫特征或模型的输入，因变量就叫预测目标或模型输出
  3. 机器学习通常使用向量兴化市表示多元模型，叫特征向量，特征向量所包含的特征项的个数就叫做维度
  4. 列向量作为向量的原始形式，用向量转置作为横向量。

1.4.3 用矩阵描述问题及算法过程¶

主要是介绍了用矩阵描述损失函数并对其求导获得最优参数的过程。 其中涉及了一些数学推导，有兴趣可以之后再草稿纸上推导。

1.5 机器学习问题¶

（1）三种机器学习问题¶

  1. 回归问题：根据样本数据建立y与x之间的关系，y为连续值
  2. 分类问题：与回归问题类似，y为离散值
  3. 聚类问题：与分类问题类似，唯一区别是样本数据集里面没有类别标注yi，无监督学习。

（2）机器学习教学三个层面¶

  1. 模型设计层面：提出新模型、新算法
  2. 模型变成方面：人家给了算法，你编程实现
  3. 应用编程方面：用人家写好的程序解决问题

2.2.3 特征选择¶

  1. 假设房价预测模型f(x)是一个线性模型
  2. 线性模型中的特征应当尽量与预测目标线性相关，否则就属于无用特征
  3. 线性模型的各项特征项之间应当尽量不相关，否则属于冗余特征
  4. 一般使用皮尔逊相关系数来衡量两个随机变量X与Y之间的线性相关程度

【注意】相关系数的表达式

  1. 相关系数的绝对值越大，则相关程度越大
  2. 若相关系数的绝对值等于1，则表示两个随机变量存在线性关系

2.2.5 数值型特征的标准化¶

数据集不同特征值取值范围差距大，不同量纲也会影响取值范围。
这种差异会造成特征项之间的不平等。
为了防止这种不平等现象被带入机器学习模型，在数据预处理环节应尽量消除量纲，统一不同特征项的取值范围，这就是数据标准化。

2.3.2 模型训练与梯度下降法【GD】¶

  1. 损失函数：实际上就是模型再训练集上的目标预测值与真实值之间的误差，写成函数形式。
  2. 学习算法：获取使得损失函数最小的参数的算法是学习算法，即优化算法。
  3. 梯度下降算法：一种优化算法，$\omega^{k} = \omega^{k-1}-a*\nabla(\omega^{k-1})$
  4. 梯度下降算法的条件：函数一阶可导
  5. 梯度下降算法还有变形：随机梯度下降算法（SGD）、小批量梯度下降算法（MBGD）：算误差所用的数据点数不一样：随机一个 or 随机多个 or 全部

定理 2-1：函数$f(x_1,x_2)$在$v$方向上的方向导数为$\frac{\partial f(x_1,x_2)}{\partial v} = \frac{\partial{f}}{x_1}cos\theta + \frac{\partial{f}}{x_2}sin\theta$

2.4 正则化¶

线性方程组:Ax=b

  1. A是非奇异的（可逆）：则方程有唯一解
  2. A是奇异的（不可逆、不满秩）：则有无穷多解
  3. A是近似奇异的(奇异值分解后，其奇异值近似等于0)：方程的解不稳定:右边的b变动只是很微小的变动就能引起解的巨大变动

不适定问题：解不唯一或者不稳定的方程定解问题

2.4.1 正则化方法¶

为了避免训练获得的参数对训练集数据太过敏感，可以使用正则化方法

奥卡姆剃刀(Occam`s razor):无从选择时应选择简单的

正则化就是在损失函数的基础上增加一个限制参数$\omega$取值过大的项，叫做正则化项或者惩罚项

概念：

  1. 损失函数
  2. L1范数条件
  3. L2范数条件
  4. 拉格朗日乘子法将单纯的优化问题变为有约束的最优化问题

坐标下降算法

梯度下降算法要求描述误差的函数是一阶可导的，岭回归使用的是L2范数，是可导的。但是LASSO回归使用的是L1范数，其中包含绝对值函数，其在最小值点是不可导的，不能 采用梯度下降法优化参数。故而，使用坐标下降算法。

坐标下降算法是选取一个参数初值，对参数矢量的参数依次进行迭代。
